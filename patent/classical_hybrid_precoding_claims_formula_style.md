# 权利要求书（公式增强版草案）

发明名称：一种面向多用户MIMO系统的结构化混合预编码方法

1. 一种面向多用户MIMO系统的结构化混合预编码方法，其特征在于，应用于基站下行发送场景，设基站发射天线数为 N_t，用户数为 K，第 k 个用户的接收天线数为 N_r，系统射频链路数为 N_RF，每个用户的数据流数为 d，第 k 个用户的下行信道矩阵为 H_k ∈ C^(N_r×N_t)，则所述方法包括如下步骤：

S1，获取各用户的预编码设计信道矩阵 H_k^des，并建立多用户下行发送模型

y_k = H_k F_RF F_BB s + n_k,    k = 1, 2, ..., K，

其中，F_RF ∈ C^(N_t×N_RF) 为共享模拟预编码矩阵，F_BB ∈ C^(N_RF×Kd) 为数字预编码矩阵，s 为发送符号向量，n_k 为噪声向量；

S2，对各用户的预编码设计信道矩阵 H_k^des 分别进行奇异值分解

H_k^des = U_k Σ_k V_k^H，

并从各用户的右奇异向量矩阵 V_k 中提取候选列向量，构造相位源矩阵 C；

S3，对所述相位源矩阵 C 进行逐元素相位提取，构造满足恒模约束的共享模拟预编码矩阵

F_RF = (1 / √N_t) exp(j·angle(C(:, 1:N_RF)))；

S4，构造各用户的等效信道矩阵

G_k = H_k^des F_RF；

S5，对第 k 个用户，将其余用户的等效信道矩阵按行堆叠得到干扰矩阵

G_-k = [G_1^T, ..., G_(k-1)^T, G_(k+1)^T, ..., G_K^T]^T，

求取所述干扰矩阵的零空间基 N_k = null(G_-k)，并构造降维等效信道矩阵

Ĝ_k = G_k N_k；

S6，对所述降维等效信道矩阵 Ĝ_k 进行结构化数字预编码设计，得到用户级数字预编码块 F_k，并将各用户数字预编码块拼接为

F_BB = [N_1 F_1, N_2 F_2, ..., N_K F_K]；

S7，对混合预编码矩阵 F = F_RF F_BB 进行总功率归一化

F_BB <- √(P / ||F_RF F_BB||_F^2) · F_BB，

其中 P 为预设数字功率约束；

S8，基于接收端性能指标对不同结构化数字预编码方案对应的混合预编码候选进行评价，并选取目标混合预编码矩阵。

2. 根据权利要求1所述的方法，其特征在于，步骤 S1 中所述预编码设计信道矩阵 H_k^des 的获取方式包括以下至少一种：

(1) 理想信道状态信息条件下，

H_k^des = H_k；

(2) 误差信道状态信息条件下，

H_k^des = H_k + E_k，

其中 E_k 为满足预设均方误差水平的复高斯误差矩阵；

(3) 导频辅助信道估计条件下，根据导频观测和信道统计先验获得 H_k^des。

3. 根据权利要求2所述的方法，其特征在于，所述导频辅助信道估计条件下，根据导频观测和信道统计先验获得 H_k^des，包括：

构造重复单位阵形式的导频矩阵 P_t，

P_t = [I_(N_t), I_(N_t), ..., I_(N_t)]，

其导频长度 L_p 满足 L_p ≥ N_t 且 L_p 为 N_t 的整数倍；

对离线生成的信道样本 H^(m) 进行向量化，令

h^(m) = vec(H^(m))，

并计算样本均值与样本协方差矩阵

h_bar = (1 / M) Σ_(m=1)^M h^(m)，

C_h = (1 / M) Σ_(m=1)^M (h^(m) - h_bar)(h^(m) - h_bar)^H；

对样本协方差矩阵施加对角加载，得到

C_h^dl = C_h + ε·max(tr(C_h) / D_h, 10^(-12))·I，

其中 ε 为对角加载系数，D_h 为信道向量维度；

根据导频观测

Y_k = H_k P_t + N_k，

建立观测矩阵

A = kron(P_t^T, I_(N_r))，

并采用最小均方误差估计得到

h_k^des = h_bar + C_h^dl A^H (A C_h^dl A^H + σ_n^2 I)^(-1) (y_k - A h_bar)，

再将 h_k^des 重构为矩阵形式 H_k^des。

4. 根据权利要求1所述的方法，其特征在于，步骤 S2 和步骤 S3 中构造共享模拟预编码矩阵时，对每个用户提取 V_k 中前 min(N_RF, N_r) 列或前 min(N_RF, d) 列右奇异向量作为候选列向量；当所有用户候选列向量拼接后的列数小于 N_RF 时，引入离散傅里叶变换导向向量 D 进行补列，使得

C = [V_1^sel, V_2^sel, ..., V_K^sel, D]，

并截取前 N_RF 列生成共享模拟预编码矩阵。

5. 根据权利要求1所述的方法，其特征在于，步骤 S6 中，所述结构化数字预编码设计至少包括以下三种方式中的一种：

(1) 奇异值分解 SVD 方式：对降维等效信道矩阵进行奇异值分解

Ĝ_k = Û_k Σ̃_k Ṽ_k^H，

并取

F_k^SVD = Ṽ_k(:, 1:d)；

(2) 几何均值分解 GMD 方式：在 Ṽ_k(:, 1:d) 的基础上施加右旋转矩阵 Ω_k，得到

F_k^GMD = Ṽ_k(:, 1:d) Ω_k，

使得等效上三角矩阵 R_k 的对角元素满足近似相等；

(3) 统一信道分解 UCD 方式：根据 Σ̃_k 与噪声加载因子 α_k 构造增广信道并生成 UCD 旋转矩阵，得到

F_k^UCD = Ṽ_k(:, 1:d) Λ_k^(1/2) Ω_k^ucd，

其中 Λ_k 为功率加载对角矩阵。

6. 根据权利要求5所述的方法，其特征在于，所述 GMD 方式中，设降维等效信道矩阵的前 d 个奇异值为 σ_(k,1), σ_(k,2), ..., σ_(k,d)，则对应目标几何均值为

r̄_k = (Π_(i=1)^d σ_(k,i))^(1/d)，

通过对右奇异向量施加级联旋转，使降维等效信道变换为上三角等效信道

B_k = Q_k^H Ĝ_k F_k^GMD = R_k，

其中 R_k 为上三角矩阵，且其对角元素满足

r_(k,1,1) ≈ r_(k,2,2) ≈ ... ≈ r_(k,d,d) ≈ r̄_k。

7. 根据权利要求6所述的方法，其特征在于，所述 GMD 方式对应的发送端采用 Tomlinson-Harashima 预编码递推，设对角归一化后的上三角矩阵为 B_k^(1)，其元素为 b_(k,i,j)，发送符号为 s_(k,i)，则第 i 层递推符号为

u_(k,i) = Mod( s_(k,i) - Σ_(j=i+1)^d b_(k,i,j) u_(k,j) )，

其中 Mod(·) 表示对实部和虚部分别执行居中模运算；最终发射向量为

x_k = F_k^GMD u_k；

接收端对接收信号执行对角均衡和模折叠，并基于模折叠后的符号计算比特交织编码调制广义互信息或误码率。

8. 根据权利要求5所述的方法，其特征在于，所述 UCD 方式中，设降维等效信道矩阵的前 d 个奇异值为 σ_(k,1), σ_(k,2), ..., σ_(k,d)，功率加载系数为 p_(k,1), p_(k,2), ..., p_(k,d)，则加载后的增广奇异值为

λ_(k,i) = √( p_(k,i) σ_(k,i)^2 + α_k )，    i = 1, 2, ..., d，

并定义统一信道分解目标对角值为

λ̄_k = exp( (1 / d) Σ_(i=1)^d ln λ_(k,i) )；

再通过统一信道分解生成旋转矩阵 Ω_k^ucd，使对应上三角增广等效信道的对角元素趋于 λ̄_k，并由此得到各数据流近似一致的目标信噪比。

9. 根据权利要求8所述的方法，其特征在于，所述功率加载系数 p_(k,i) 采用类注水分配方式获得，对于处于激活状态的数据流集合 S_k，其功率分配满足

p_(k,i) = max( μ_k - α_k / σ_(k,i)^2, 0 )，    i ∈ S_k，

其中 μ_k 为注水水位；对非激活状态的数据流分配零功率或预设最小加载功率。

10. 根据权利要求1所述的方法，其特征在于，步骤 S8 中的接收端性能评价指标包括和速率、误码率以及比特交织编码调制广义互信息中的至少一种；当采用和速率作为评价指标时，所述和速率表示为

R_sum = Σ_(k=1)^K Σ_(i=1)^d log_2(1 + ρ_(k,i))，

其中 ρ_(k,i) 为第 k 个用户第 i 个数据流的等效信噪比。

11. 根据权利要求10所述的方法，其特征在于，当采用比特交织编码调制广义互信息作为评价指标时，对第 l 个比特位的对数似然比 L_(k,i,l) 计算为

L_(k,i,l) = log Σ_(x∈X_l^1) exp( -|z_(k,i)-x|^2 / σ_n^2 )
            - log Σ_(x∈X_l^0) exp( -|z_(k,i)-x|^2 / σ_n^2 )，

其中 z_(k,i) 为经均衡后的接收符号，X_l^1 和 X_l^0 分别表示第 l 个比特位取值为 1 和 0 的候选星座点集合；并基于相同的调制符号样本和相同的噪声样本对不同混合预编码候选进行蒙特卡洛评价。

12. 一种用于实现权利要求1至11任一项所述方法的结构化混合预编码装置，其特征在于，包括：

信道获取模块，用于获取多用户下行信道矩阵集合及系统参数；
信道状态信息确定模块，用于获得预编码设计信道矩阵 H_k^des；
共享模拟预编码构建模块，用于构建满足恒模约束的共享模拟预编码矩阵 F_RF；
块对角化降维模块，用于构建各用户零空间基 N_k 和降维等效信道矩阵 Ĝ_k；
结构化数字预编码构建模块，用于构建 SVD、GMD 或 UCD 数字预编码块 F_k；
性能评价与选择模块，用于对不同混合预编码候选进行接收端性能评价并选择目标混合预编码矩阵。

13. 一种电子设备，其特征在于，包括处理器和存储器，所述存储器中存储有计算机程序，所述计算机程序被所述处理器执行时，使所述电子设备执行权利要求1至11任一项所述的方法。

14. 一种计算机可读存储介质，其上存储有计算机程序，其特征在于，所述计算机程序被处理器执行时，实现权利要求1至11任一项所述的方法。

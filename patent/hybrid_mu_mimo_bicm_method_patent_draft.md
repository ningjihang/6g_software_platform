# 一种面向多用户 MIMO-BICM 系统的混合预编码选择方法

## 摘要

本发明公开了一种面向多用户 MIMO-BICM 系统的混合预编码选择方法。该方法首先获取多用户下行信道矩阵，在理想信道状态信息条件下直接基于所述下行信道矩阵进行预编码设计，在非理想信道状态信息条件下基于导频观测和离线信道样本估计得到的信道均值及全协方差矩阵，对发射端信道状态信息进行全协方差 LMMSE 估计；随后根据发射端可见信道为各用户构建满足恒模约束的共享模拟预编码矩阵，并在所述模拟预编码矩阵作用后的等效信道上构建每个用户的块对角化零空间；再针对各用户的降维等效信道生成结构化数字预编码候选，所述候选包括 SVD 候选、GMD 候选、UCD 候选以及位于 SVD 与 GMD 之间的过渡型候选，其中所述 GMD 候选包括基于几何均值分解得到的上三角等效信道以及基于所述上三角等效信道执行的 Tomlinson-Harashima 预编码发送递推；最后在相同调制阶数、信噪比和 Monte Carlo 样本条件下，对各候选进行 BICM 广义互信息和误码率评价，并根据评价结果选择混合预编码矩阵。本发明能够在有限星座输入和非理想 CSI 条件下，提高多用户 MIMO-BICM 系统的预编码选择可靠性，并降低单纯依赖固定 SVD 或固定 GMD 方案带来的性能波动。

## 权利要求书

### 权利要求 1

一种面向多用户 MIMO-BICM 系统的混合预编码选择方法，其特征在于，包括如下步骤：

S1，获取包含多个用户的下行信道矩阵集合，并确定系统参数，所述系统参数包括发射天线数、接收天线数、用户数、射频链路数、每用户数据流数、调制阶数和信噪比；

S2，基于所述下行信道矩阵集合获得发射端用于预编码设计的信道状态信息；

S3，根据所述信道状态信息构建共享模拟预编码矩阵，所述共享模拟预编码矩阵满足恒模约束；

S4，将各用户的下行信道矩阵与所述共享模拟预编码矩阵相乘，得到各用户的等效信道矩阵；

S5，针对每个用户，根据其他用户的等效信道矩阵构建块对角化零空间，并将该用户的等效信道矩阵投影至所述块对角化零空间，得到该用户的降维等效信道矩阵；

S6，基于所述降维等效信道矩阵生成多个结构化数字预编码候选，并将各用户的数字预编码候选组合成多用户数字预编码矩阵；

S7，对由所述共享模拟预编码矩阵和所述多用户数字预编码矩阵形成的混合预编码候选进行接收端性能评价；

S8，根据所述接收端性能评价结果，从所述混合预编码候选中选择目标混合预编码矩阵。

## 说明书

### 技术领域

本发明涉及无线通信技术领域，尤其涉及一种面向多用户 MIMO-BICM 系统的混合预编码选择方法。

### 背景技术

多输入多输出系统能够利用多天线空间自由度提升无线通信系统的频谱效率。对于毫米波通信、大规模天线阵列和多用户下行传输场景，完全数字预编码需要为每根天线配置独立射频链路，硬件复杂度和功耗较高。因此，混合预编码结构被广泛采用，其通常由模拟预编码矩阵和数字预编码矩阵共同组成。

在采用比特交织编码调制的 MIMO 系统中，实际传输常使用有限阶 QAM 星座。与高斯输入容量优化不同，有限星座输入下的预编码设计不仅受信道奇异值影响，还受调制阶数、接收检测方式、误码传播和比特级互信息影响。传统 SVD 预编码在低信噪比或强弱子信道差异明显时具有较好稳定性，而 GMD、UCD 及其变体结构能够在特定高信噪比区间改善弱数据流性能，但在中等信噪比和非理想信道状态信息条件下可能出现性能波动。

此外，多用户系统还需要同时考虑用户间干扰抑制和每个用户内部多数据流检测。若仅根据发射端信道分解结果选择预编码矩阵，而不结合有限星座输入下的接收端 BICM 广义互信息评价，则可能导致所选预编码在实际接收链路中并非最优。因此，需要一种能够结合非理想 CSI、混合预编码结构、多用户干扰抑制、结构化数字预编码候选以及 GMD 方案对应的 THP 感知接收评价的混合预编码选择方法。

### 发明内容

#### 要解决的技术问题

本发明旨在解决如下技术问题：

1. 在多用户 MIMO-BICM 系统中，有限星座输入下单纯采用固定 SVD、GMD 或 UCD 预编码难以在不同信噪比区间和非理想 CSI 条件下保持稳定性能；

2. 传统混合预编码设计通常将模拟预编码、数字预编码和接收端评价分开处理，导致预编码选择准则与实际 BICM 接收性能不完全一致；

3. 在包含 THP 发送递推的 GMD 结构化预编码中，若接收端仍采用普通线性或 SIC 评价准则，会低估或误判 THP 模运算和周期复制星座对比特级互信息的影响。

#### 技术方案

为解决上述问题，本发明提供一种面向多用户 MIMO-BICM 系统的混合预编码选择方法。该方法包括如下流程。

首先，获取多用户下行信道矩阵集合，并确定系统参数。对于第 k 个用户，其下行信道矩阵记为 H_k。系统参数包括发射天线数 N_t、接收天线数 N_r、射频链路数 N_RF、用户数 K、每用户数据流数 d、调制阶数 M 以及信噪比。

其次，获得发射端用于预编码设计的信道状态信息。在一种实施方式中，先由信道模型生成多个离线信道样本，将每个信道样本按列向量化，估计信道向量均值和全协方差矩阵，并对全协方差矩阵施加对角加载。随后采用重复单位矩阵导频获得导频观测，并基于所述均值、全协方差矩阵和导频观测执行 LMMSE 信道估计，得到发射端 CSI。

再次，构建共享模拟预编码矩阵。对每个用户的 CSI 执行奇异值分解，提取与每用户数据流数对应的右奇异向量相位，组合形成满足恒模约束的共享模拟预编码矩阵 F_RF。然后得到每个用户的等效信道 H_k F_RF。

随后，针对每个用户构建块对角化零空间。对于目标用户 k，将其他用户的等效信道矩阵堆叠形成干扰信道矩阵，并计算该干扰信道矩阵的零空间基。将目标用户等效信道投影至该零空间，得到降维等效信道。

接着，对每个用户的降维等效信道生成多个结构化数字预编码候选。候选可以包括 SVD 数字预编码、GMD 数字预编码、UCD 数字预编码以及位于 SVD 与 GMD 之间的过渡型数字预编码，其中 GMD 数字预编码包括基于上三角等效信道的 THP 发送符号递推。将各用户候选组合为完整多用户数字预编码矩阵 F_BB，并对 F_RF F_BB 执行总功率归一化。

最后，在相同符号样本和噪声样本下，对不同混合预编码候选进行接收端性能评价。对于 SVD 候选、UCD 候选和过渡型候选，基于接收端等效信号计算 BICM 广义互信息和误码率。对于 GMD 候选，先根据上三角等效信道执行 THP 发送端反馈抵消和模运算，再在接收端进行对角等化和模折叠，并枚举周期复制星座点计算比特对数似然比，由此估计 THP 感知 BICM 广义互信息和误码率。根据评价结果选择目标混合预编码矩阵。

#### 有益效果

与现有技术相比，本发明至少具有如下有益效果：

1. 将全协方差 LMMSE 信道估计引入混合预编码选择流程，使预编码设计能够利用信道相关性先验，改善非理想 CSI 条件下的预编码可靠性；

2. 在共享模拟预编码和多用户块对角化的基础上同时构建 SVD、GMD、UCD 以及位于 SVD 与 GMD 之间的过渡型结构化候选，避免在所有信噪比区间固定采用单一预编码结构；

3. 对包含 THP 发送递推的 GMD 候选采用模折叠和周期复制星座的比特级评价方式，使预编码选择准则更贴近有限星座 MIMO-BICM 的实际接收性能；

4. 在不同候选之间使用相同符号样本和噪声样本进行 Monte Carlo 评价，减少候选比较中的随机方差，提高选择结果稳定性；

5. 通过候选排序和可选的联合软优化，在性能和计算复杂度之间取得平衡。

### 具体实施方式

以下结合一个具体多用户下行传输场景说明本发明的实施方式。该实施方式用于解释本发明，不构成对保护范围的限制。

设系统包括 K 个用户，基站侧发射天线数为 N_t，射频链路数为 N_RF，每个用户接收天线数为 N_r，每个用户的数据流数为 d。第 k 个用户的信道矩阵为 H_k，发送端采用有限阶 QAM 星座并进行 BICM 传输。

#### 1. 信道状态信息获取

在一种实施方式中，先基于目标信道模型生成 N_cov 个信道样本。对每个样本 H^(n) 按列向量化为 h^(n)=vec(H^(n))。计算样本均值：

```text
bar_h = (1 / N_cov) * sum_n h^(n)
```

并计算样本协方差：

```text
C_h = (1 / N_cov) * sum_n (h^(n)-bar_h)(h^(n)-bar_h)^H
```

为提高数值稳定性，对 C_h 施加对角加载：

```text
C_h_dl = C_h + epsilon * max(tr(C_h)/D_h, 1e-12) * I
```

其中 D_h 为信道向量维度，epsilon 为对角加载系数。

构建重复单位矩阵导频 P，其长度 L 满足 L >= N_t 且 L 为 N_t 的整数倍。对于每个用户，根据导频观测：

```text
Y_k = H_k P + N_k
```

构建观测矩阵 A = kron(P^T, I)，并通过 LMMSE 公式：

```text
hat_h_k = bar_h + C_h_dl A^H (A C_h_dl A^H + sigma_n^2 I)^(-1) (y_k - A bar_h)
```

得到估计信道 hat_H_k。

#### 2. 共享模拟预编码构建

对每个用户的估计信道 hat_H_k 进行奇异值分解，选取前 d 个右奇异向量，并提取相位。将多个用户的相位向量组合为共享模拟预编码矩阵 F_RF，并使其满足如下恒模约束：

```text
|[F_RF]_{i,j}| = 1 / sqrt(N_t)
```

由此获得各用户的等效信道：

```text
G_k = hat_H_k F_RF
```

#### 3. 块对角化降维信道构建

对目标用户 k，将其他用户的等效信道堆叠为：

```text
G_-k = [G_1^T, ..., G_{k-1}^T, G_{k+1}^T, ..., G_K^T]^T
```

计算 G_-k 的零空间基 N_k，并得到目标用户降维等效信道：

```text
tilde_G_k = G_k N_k
```

该处理使每个用户的数字预编码块被限制在其他用户等效信道的零空间内，从而降低用户间干扰。

#### 4. 结构化数字预编码候选生成

对 tilde_G_k 执行奇异值分解：

```text
tilde_G_k = U_k Sigma_k V_k^H
```

根据不同策略生成数字预编码候选。

对于 SVD 候选，直接采用 V_k 的前 d 列作为用户局部数字预编码基。

对于 GMD 候选，根据奇异值计算几何均值目标对角增益，并通过右酉旋转将对角信道变换为对角元素近似相等的上三角等效信道，并基于所述上三角等效信道执行 THP 发送符号递推。

对于 UCD 候选，先根据降维等效信道和噪声功率构造增广信道矩阵，并结合统一信道分解目标生成局部数字预编码基。

对于过渡型候选，对前部数据流保留对应的 SVD 目标增益，对其余数据流采用剩余目标增益的几何均值，并通过右酉旋转将所述降维等效信道变换为与目标对角增益对应的上三角等效信道。

对于每种候选，将用户局部预编码基左乘其块对角化零空间基 N_k，得到该用户数字预编码块。将所有用户的数字预编码块按列拼接，得到多用户数字预编码矩阵 F_BB，并对 F_RF F_BB 进行总功率归一化。

#### 5. GMD 候选中的 THP 发送递推

对于 GMD 候选，对每个用户的等效上三角信道 B_k，先按其对角元素归一化得到反馈矩阵。对于第 i 层数据流，从后向前递推：

```text
u_i = s_i - sum_{j=i+1}^d b_{i,j} u_j
```

随后对 u_i 执行居中模运算，得到 THP 发送符号。该过程能够在发送端预先抵消用户内部上三角串扰。

#### 6. 接收端评价与候选选择

对于每个混合预编码候选，使用相同的符号样本和噪声样本进行评价。对 SVD 候选、UCD 候选和过渡型候选，基于接收端投影后的信号计算 BICM 广义互信息和误码率。

对 GMD 候选，接收端先进行对角等化：

```text
z_i = y_i / b_{i,i}
```

然后对 z_i 执行模折叠。为计算比特对数似然比，枚举原始星座点在模域中的周期复制点，并计算：

```text
LLR_l = log sum_{x in X_l^1} exp(-|z_i - x|^2 / sigma^2)
        - log sum_{x in X_l^0} exp(-|z_i - x|^2 / sigma^2)
```

其中 X_l^1 和 X_l^0 分别表示第 l 个比特为 1 和 0 的周期复制星座集合。根据 LLR 估计每个用户的 BICM 广义互信息，并计算多用户和速率。

最终，选择使多用户和速率最大的候选作为目标混合预编码矩阵。在一种可选实施方式中，还可结合用户间公平性、用户间泄漏功率和误码率对候选进行综合排序。

#### 7. 可选联合软优化

在一种可选实施方式中，以上述结构化候选中评价最优的候选作为初始化，对共享模拟预编码矩阵和数字预编码矩阵进行联合软优化。优化变量包括模拟预编码矩阵相位和数字预编码块。优化目标为 THP 感知 BICM 广义互信息，同时加入用户间干扰惩罚项：

```text
maximize  R_GMI - lambda * P_interference
```

其中 R_GMI 表示多用户 BICM 广义互信息，P_interference 表示用户间干扰泄漏功率，lambda 为惩罚权重。优化完成后，重新进行接收端评价并输出最终目标混合预编码矩阵。

```text

每用户接收天线数 N_r = 4
射频链路数 N_RF = 8

```
